# Three.js Lights
Lighting in Three.js and React-Three-Fiber is essential for creating realistic and visually appealing 3D scenes. Various types of lights provide different effects and interactions with materials.


## 1. Light (Base Class)
The `Light` class is the base for all lights in Three.js. It is not used directly.

```javascript
const light = new THREE.Light(0xffffff, 1);
```

## 2. AmbientLight
A global light that affects all objects equally.

###### Three.js Example
```javascript
const light = new THREE.AmbientLight(0xffffff, 0.5);
scene.add(light);
```

###### React-Three-Fiber Example
```jsx
<ambientLight intensity={0.5} />
```

<AmbientLight />


## 3. DirectionalLight
A light source that acts like the sun, with parallel rays.

###### Three.js Example
```javascript
const light = new THREE.DirectionalLight(0xffffff, 1);
light.position.set(5, 10, 5);
scene.add(light);
```

###### React-Three-Fiber Example
```jsx
<directionalLight position={[5, 10, 5]} intensity={1} />
```

<DirectionalLight />


## 4. HemisphereLight
A gradient-based light that simulates a sky dome.

###### Three.js Example
```javascript
const light = new THREE.HemisphereLight(0x4040ff, 0x80ff80, 0.6);
scene.add(light);
```

###### React-Three-Fiber Example
```jsx
<hemisphereLight groundColor="green" intensity={0.6} />
```

<HemisphereLight />


## 5. LightProbe
Used for advanced global illumination techniques.

###### Three.js Example
```javascript
const lightProbe = new THREE.LightProbe();
scene.add(lightProbe);
```

###### React-Three-Fiber Example
```jsx
<lightProbe />
```

<LightProbe />


## 6. PointLight
A light that emits in all directions from a single point.

###### Three.js Example
```javascript
const light = new THREE.PointLight(0xff0000, 1, 10);
light.position.set(2, 3, 2);
scene.add(light);
```

###### React-Three-Fiber Example
```jsx
<pointLight position={[2, 3, 2]} intensity={1} distance={10} color="red" />
```

<PointLight />


## 7. RectAreaLight
A light that emits from a rectangular area.

###### Three.js Example
```javascript
const rectLight = new THREE.RectAreaLight(0xffffff, 2, 4, 2);
scene.add(rectLight);
```

###### React-Three-Fiber Example
```jsx
<rectAreaLight width={4} height={2} intensity={2} color="white" />
```

<RectAreaLight />


## 8. SpotLight
A light that emits in a specific direction with a conical effect.

###### Three.js Example
```javascript
const light = new THREE.SpotLight(0xffffff, 1);
light.position.set(5, 10, 5);
scene.add(light);
```

###### React-Three-Fiber Example
```jsx
<spotLight position={[5, 10, 5]} intensity={1} />
```

<SpotLight />


## Summary
- **Light**: Base class, not used directly.
- **AmbientLight**: Provides uniform lighting across the scene.
- **DirectionalLight**: Mimics sunlight with parallel rays.
- **HemisphereLight**: Simulates sky dome lighting.
- **LightProbe**: Used for global illumination.
- **PointLight**: Emits light in all directions from a single point.
- **RectAreaLight**: Emits light from a rectangular area.
- **SpotLight**: Emits a focused beam in one direction.
